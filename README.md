# Random Quotes

A web service responses with random quotes.

## Request

Command line:

```bash
$ curl https://random-quotes.herokuapp.com
```

Browsers:

```javascript
$.get('https://random-quotes.herokuapp.com/', function (data) {
  console.log(data)
})
```
